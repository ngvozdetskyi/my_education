## Running the app
To run app follow these steps:
1. You should have docker.
2. Create env files for different stages in config directory(There examples).
3. Run next commands' sequentially.
```bash
# development
$ npm run start:dev
$ npm run start
```
## Stopping the app
```bash
# development
$ npm run down:dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
